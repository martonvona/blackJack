##Black Jack kártya játék

####Rendszer követelmények:
- maven 3.x
- jdk 1.8.0_77 

####Futtatás:

- **1.lépés:**
Fordítás

        > mvn package

- **2.lépés:** A hsqldb adatbázis szerver futtatása

        > mvn exec:java

- **3.lépés:** A jaték indítása 

        > java -jar target/black-jack-1.0-jar-with-dependencies.jar
